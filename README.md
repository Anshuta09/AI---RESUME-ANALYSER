# AI---RESUME-ANALYSER
AI Resume Analyzer built with TypeScript and TSX that analyzes resume content and provides AI-driven feedback through a modern, component-based UI. Focuses on frontend architecture, type safety, and structured AI response handling using the Node.js ecosystem.
